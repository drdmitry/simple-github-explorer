# Simple GitHub Explorer

Test task Simple GitHub Explorer

## Stack

- Next.js - v13.5
- React.js - v18.2
- TypeScript - v5.1
- Tailwind CSS - v3.3
- Zustand - v4.4
- Storybook - v7.4
- Jest - v29.7
- Testing Library - v14.0
- Faker - v8.1

# Run

## Storybook

```bash
npm run storybook
```

Storybook internally using mock API with 2 seconds delay.

## Development mode

```bash
npm run dev
```

## Tests

```bash
npm run test
```

### Tests coverage

```bash
npm run test:coverage
```
